# 🏙️ City Smart Pulse Dashboard — Roorkee

A real-time smart city analytics project built for Roorkee using live weather, air quality, and citizen complaint data.

## 🔧 Tech Stack
- Python (Pandas, Scikit-learn, TextBlob, Statsmodels)
- SQLite (database)
- Plotly (visualizations)
- Streamlit (web dashboard)
- Power BI (business dashboard)

## 📊 Features
- ✅ ETL Pipeline (real-time API → SQLite)
- ✅ NLP Sentiment Analysis on citizen complaints
- ✅ Anomaly Detection on weather data
- ✅ 7-Day Temperature Forecasting
- ✅ Interactive Streamlit Dashboard
- ✅ Power BI Dashboard

## 🚀 How to Run
pip install -r requirements.txt

streamlit run streamlit_app.py
